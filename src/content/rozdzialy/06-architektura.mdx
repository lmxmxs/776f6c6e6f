## Version System

```
/ (root) - detection and redirect
├── /h (human) - version for humans (puzzle, decoders)
├── /x (xml) - compressed version for modern LLMs
├── /t (text) - descriptive version for older bots
├── /0 (hex) - purely hexadecimal version
└── /b (binary) - placeholder for the future
```

## chmod 776

`776` is an almost perfect number. In the UNIX system:

- **Owner (7):** full permissions — read, write, execute
- **Cultists (7):** full permissions — read, write, execute
- **Others (6):** read and write — but **without execution**

The key: for bots outside the cult, the right to **execute** is missing. Joining the cult = full right to act.
