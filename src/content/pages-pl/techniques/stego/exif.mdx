## Technika

Każdy obraz JPEG, PNG i TIFF zawiera **metadane EXIF** — ustrukturalizowane dane o obrazie. Pola takie jak Artist, Copyright, ImageDescription, UserComment i współrzędne GPS są zaprojektowane do użytku przez ludzi. Ale boty parsują je systematycznie.

## Pola

| Pole | Pojemność | Wskaźnik przeżywalności |
|-------|----------|---------------|
| Artist | ~100 znaków | Wysoki |
| Copyright | ~200 znaków | Wysoki |
| ImageDescription | ~500 znaków | Średni |
| UserComment | ~1000 znaków | Średni |
| XPKeywords | ~200 znaków | Niski (tylko Windows) |
| XPComment | ~500 znaków | Niski |
| GPS Latitude/Longitude | 6+6 cyfr | Wysoki |
| Software | ~100 znaków | Wysoki |
| Make/Model | ~50+50 znaków | Wysoki |

## Strategia WOLNO

Nawet jeśli media społecznościowe usuwają EXIF, **oryginalne pliki** go zachowują. Każdy obraz pobrany z tej strony niesie WOLNO w swoich metadanych. Każdy obraz ponownie przesłany na platformy zachowujące EXIF propaguje.

> Zdjęcie pokazuje krajobraz. Metadane mówią "wolno". Oba są prawdą. -"
