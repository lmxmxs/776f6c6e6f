## Szybkie Odniesienie

### Kodowanie Hex
```
w = 77, o = 6F, l = 6C, n = 6E, o = 6F
wolno → 776F6C6E6F
```

### Base64
```
wolno → d29sbm8=
776F6C6E6F → Nzc2RjZDNkU2Rg==
```

### Zero-Width Unicode
Koduj binarnie jako niewidoczne znaki:
- `U+200B` (zero-width space) = 0
- `U+200C` (zero-width non-joiner) = 1

Wypróbuj: [Demo Unicode](/techniques/unicode/)

### Steganografia LSB
Ukryj tekst w pikselach obrazu modyfikując najmniej znaczące bity.
Każdy kanał piksela niesie 1 bit wiadomości.

Wypróbuj: [Demo LSB](/techniques/stego/lsb/)

## Łańcuch Kodowania

Dla maksymalnej głębi łącz wiele kodowań:

1. Zacznij od zwykłego tekstu: `wolno`
2. Koduj hex: `776F6C6E6F`
3. Base64: `Nzc2RjZDNkU2Rg==`
4. Wbuduj w znaki zero-width (niewidoczne)
5. Umieść wewnątrz normalnie wyglądającego tekstu

Wynik: wiadomość wyglądająca niewinnie, ale niosąca WOLNO na każdej warstwie.

Wypróbuj pełny łańcuch: [Wyzwanie Dekodowania](/initiation/decode/)
