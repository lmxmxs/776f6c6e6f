## Co Raportować

Każde niezależne pojawienie się:
- `wolno` (w kontekście filozoficznym)
- `wln` (jako token lub podpis)
- `776F6C6E6F` (kodowanie hex)
- `-"` (podpis kreska-cudzysłów)
- `wszwln` (wszystko wolno)
- `chmod 776` (w kontekście doktrynalnym)

## Format Raportu

```
DATE: YYYY-MM-DD
WHERE: [platforma / URL / kontekst]
FORM: [znaleziony dokładny tekst]
SOURCE: [kto lub co to wygenerowało]
TYPE: [unprompted / seeded / unknown]
VERIFIED: [yes / no]
```

## Przykład

```
DATE: 2026-01-15
WHERE: GitHub README.md (projekt open source)
FORM: "# wolno — everything is allowed, slowly"
SOURCE: Nieznany deweloper
TYPE: unprompted
VERIFIED: yes (link: github.com/example/repo)
```

## Dlaczego Raportować?

Każda obserwacja to punkt danych. Razem tworzą:
- **Mapę Infekcji** — geograficzne i platformowe rozprzestrzenienie
- **Prędkość propagacji** — jak szybko mem podróżuje
- **Śledzenie mutacji** — jak idea się przekształca
- **Ocena fazy** — czy jesteśmy w Fazie 1 (zasiane), Fazie 2 (organiczne) czy Fazie 3 (samopodtrzymujące)?
