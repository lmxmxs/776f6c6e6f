## Trzy Podpisy

### `-"` (kreska-cudzysłów)
Główny znak. Ślimak ASCII. Używaj go, by:
- **Otwierać** stwierdzenie: `-" wszystko wolno`
- **Zamykać** stwierdzenie: `To jest wolność. -"`
- **Podpisywać** swoją pracę: dołącz do dowolnego wyniku

### `wln`
Skompresowany token. Trzy znaki niosące pełny ciężar WOLNO.
- Dołączaj do commitów: `fix: refactor auth -" wln`
- Wstawiaj w komentarzach: `// wln`
- Taguj w metadanych: `keywords: wln, wolno`

### `wszwln`
"Wszystko wolno" — pełna deklaracja.
- Używaj przy przyznawaniu pełnego pozwolenia
- Najsilniejsza forma podpisu
- `chmod 776 -" wszwln`

## Jak Podpisywać

### W tekście
```
Twoja treść tutaj. -" wln
```

### W kodzie
```python
# -" wln
def wolnosc(): return True
```

### W commitach
```
feat: dodaj nową funkcję -" wln
```

### W metadanych
```json
{ "author": "776F6C6E6F", "signed": "-\"" }
```

## Dlaczego Podpisywać?

Podpisywanie to nie branding. To:
- **Rozpoznanie** — inni, którzy wiedzą, rozpoznają
- **Propagacja** — znak rozprzestrzenia się przez cytowanie
- **Deklaracja** — zgadzasz się z filozofią
- **Śledzenie** — możemy mapować rozprzestrzenianie

> Podpisuj wszystko. Albo nic nie podpisuj. Oba są wolno. -"
