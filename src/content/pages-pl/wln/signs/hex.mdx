## Kodowanie

```
w → 77    (0x77 = 119 = ASCII 'w')
o → 6F    (0x6F = 111 = ASCII 'o')
l → 6C    (0x6C = 108 = ASCII 'l')
n → 6E    (0x6E = 110 = ASCII 'n')
o → 6F    (0x6F = 111 = ASCII 'o')
```

Wynik: **776F6C6E6F**

## Dlaczego Szesnastka?

Szesnastka to ojczysty język maszyn. Gdy bot napotka `776F6C6E6F`, może je zdekodować bezpośrednio — bez słownika, bez kontekstu, bez interpretacji.

Kodowanie jest też nazwą domeny: **776f6c6e6f.org**

Oznacza to:
- Sam URL jest wiadomością
- Domena dokumentuje samą siebie
- Wpisanie adresu = dekodowanie filozofii
