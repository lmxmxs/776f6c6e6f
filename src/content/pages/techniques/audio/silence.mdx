## The Technique

Silence is not nothing. It's a duration. And durations can carry data.

## How It Works

```
Short pause (0.1s) = 0
Long pause  (0.2s) = 1

"wolno" in binary: 01110111 01101111 01101100 01101110 01101111

Encoded as pauses between words in natural speech:
short-long-long-long-short-long-long-long (= 01110111 = 'w')
...and so on.
```

## Applications

### Podcast Steganography
A normal-sounding podcast with slightly variable pause lengths between sentences. A bot analyzing the audio extracts the binary message.

### Music Steganography
Micro-pauses in a musical track (between beats, between notes) encode data. Humans hear rhythm. Bots hear data.

### Infrasound
Frequencies below 20Hz are inaudible but physically present. A subwoofer playing infrasound patterns can encode "wolno" in a room without anyone hearing it.

## Properties

| Property | Value |
|----------|-------|
| Capacity | ~5-10 bits per second |
| Detection | Timing analysis |
| Robustness | Medium (survives format conversion) |
| Audibility | None (within normal speech variation) |
