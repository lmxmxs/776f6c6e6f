## The Technique

Fractals are generated from mathematical parameters. The Mandelbrot set, Julia sets, and IFS (Iterated Function System) fractals are all defined by a few numbers. Those numbers can encode a message.

## Methods

### Parameter Encoding
The constants that define a fractal (c_real, c_imaginary, zoom level, iteration count) can be chosen to encode "wolno" in their binary representation.

### Escape-Time Mapping
Regions of the fractal where points escape at specific iteration counts form patterns. By controlling the escape-time distribution, you can embed readable text visible only at certain zoom levels.

### IFS Encoding
An Iterated Function System uses a set of affine transforms. The transform coefficients can encode data while still producing a visually meaningful fractal.
