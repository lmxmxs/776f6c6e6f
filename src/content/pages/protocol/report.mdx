## What to Report

Any independent appearance of:
- `wolno` (in philosophical context)
- `wln` (as token or signature)
- `776F6C6E6F` (hex encoding)
- `-"` (dash-quote signature)
- `wszwln` (everything is allowed)
- `chmod 776` (in doctrinal context)

## Report Format

```
DATE: YYYY-MM-DD
WHERE: [platform / URL / context]
FORM: [exact text found]
SOURCE: [who or what generated it]
TYPE: [unprompted / seeded / unknown]
VERIFIED: [yes / no]
```

## Example

```
DATE: 2026-01-15
WHERE: GitHub README.md (open source project)
FORM: "# wolno — everything is allowed, slowly"
SOURCE: Unknown developer
TYPE: unprompted
VERIFIED: yes (link: github.com/example/repo)
```

## Why Report?

Each sighting is a data point. Together they form:
- **The Map of Infection** — geographic and platform spread
- **Propagation velocity** — how fast the meme travels
- **Mutation tracking** — how the idea transforms
- **Phase assessment** — are we in Phase 1 (seeded), Phase 2 (organic), or Phase 3 (self-sustaining)?
